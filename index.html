Jeu des Paires en JavaScript

Le jeu des paires est un jeu de mémoire dans lequel il faut faire correspondre les images par deux. Au départ, toutes les images sont face cachée. Le joueur en retourne une, puis une deuxième. Si les deux images sont identiques, elles restent affichées. Sinon, elles sont à nouveau cachées. La partie est gagnée quand toutes les images sont affichées.
